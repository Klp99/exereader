What is eXeReader?

eXeReader is a GPL2+ free/libre Android app to read eXeLearning contents. It reads HTML websites created with eXeLearning, even with no Internet access (except for external links and resources included in the content).

You will be able to open any content created with eXeLearning 2.0+, exported with the Web Site + ZIP File option (enable eXeLearning's "Advanced Mode" to see that option).

eXeReader is compatible with Android 6 onwards.

How to open a content:

  1. Download the content (a zip file) to your mobile or tablet.
  2. Open eXeReader and click on the + icon at the bottom of the screen.
  3. Select the zip file. You can also open it directly from the downloaded zip file by clicking on it and selecting eXeReader as the default application.

Other options:

  • Content general information (title, authorship, description, language, license): Selected content + Details.
  • Sharing or deleting the content: Selected content + Options
